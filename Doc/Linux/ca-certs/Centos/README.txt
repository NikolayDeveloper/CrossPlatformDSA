В данном каталоге предоставлены тестовые и боевые ca-сертификаты.


----Установка корневых и промежуточных сертификатов на ОС Linux-Centos----

1) Скопируйте сертификаты в каталог /usr/share/pki/ca-trust-source/anchors/
2) Выполните команду: update-ca-trust extract

 Для проверки - откройте файл /etc/pki/ca-trust/extracted/openssl/ca-bundle.trust.crt. В конце списка должны быть сертификаты НУЦа
