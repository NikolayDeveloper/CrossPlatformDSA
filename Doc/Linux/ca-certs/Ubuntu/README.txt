----Инструкция----
1) Скопируйте нужный архив в Ubuntu.
2) Разахивируйте его. 
3) Запустите из командной строки от имени администратора файл instal_*.sh, но для начала проверь пути, чтобы там лежали корневые сертификаты.
4)Далее проверь, что корневые сертификаты правильно установились, открой вот этом файл /etc/ssl/certs/ca-certificates.crt и посмотри в списке свои сертификаты.
